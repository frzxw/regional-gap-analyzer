# Data Directory

This directory contains all data files for the Regional Gap Analyzer.

## Directory Structure

```
data/
├── raw/          # Original, unprocessed data files
├── processed/    # Cleaned and normalized data
├── geo/          # GeoJSON and geographic boundary files
└── README.md     # This file
```

## Data Sources

**IMPORTANT:** All data in this directory is for demonstration purposes only unless explicitly marked as official.

### Raw Data (`raw/`)
- Place original CSV, Excel, or JSON files here
- Files should be named with source and date: `{source}_{indicator}_{YYYYMMDD}.csv`
- Example: `bps_gdp_per_capita_20250101.csv`

### Processed Data (`processed/`)
- Normalized and cleaned datasets ready for analysis
- Generated by data processing scripts in `backend/`

### Geographic Data (`geo/`)
- GeoJSON boundary files for provinces/regions
- Should use standard codes (ISO 3166-2)

## Data Handling Rules

1. **Never commit sensitive or personal data**
2. **Large files (>10MB) should use Git LFS or be excluded**
3. **Always document data sources and processing steps**
4. **Demo data must be clearly labeled as such**
